# Aplikacja pogodowa – React

Projekt zaliczeniowy z przedmiotu Programowanie Frontend.

## Technologie
- React
- Vite
- React Router
- Redux Toolkit
- TailwindCSS
- Axios
- OpenWeatherMap API

## Funkcjonalności
- lista miast
- szczegóły pogody i prognoza 5-dniowa
- zmiana jednostek temperatury
- wyszukiwanie miast
- ulubione miasta (Redux + localStorage)

## Uruchomienie projektu
1. `npm install`
2. zmienić rozszerzenie pliku `.env.example` na `.env`
3. `npm run dev`
